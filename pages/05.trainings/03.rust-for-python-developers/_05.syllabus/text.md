## Course syllabus

 - Writing and compilation of Rust programs.
 - Variables, mutability and data types.
 - Basic syntax:
    - functions
    - loops
    - conditionals
 - Ownership, references, and borrowing.
 - Structs.
 - Pattern matching.
 - Vectors, strings, and mappings.
 - Error handling.
 - Generic data types.
 - Traits.
 - Reference lifetimes.
 - File I/O.
 - Package management.
 - Testing.
