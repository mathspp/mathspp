## Details

Rodrigo will continue coaching us in the the use of the Textual TUI (Terminal User Interface). These are code-along-at-home exercises. At the last meeting, we started by looking at basic UI widgets and their (CSS) styling. Then, the bindings which connect Python to the front-end.

During this next session, can look forward to learning in more depth, composing the UI, making things more reactive and dynamic. Plus, the icing-on-the-cake will be learning how to use the same UI (or TUI) on the web.
